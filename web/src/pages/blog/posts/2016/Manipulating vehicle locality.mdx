# Manipulating Vehicle Locality

A while ago I was working on a system that would let me manipulate remote objects as if they were local to me.
I wanted commands like `setDamage` and `setVelocity` to work on objects that were created by the server.
Vehicles turned out to be the easiest case — Arma 3 already swaps a vehicle's _locality_ to whichever client is in the driver seat, so the driver doesn't rubber-band on their own screen.

I spent some time poking at the timing and found there's a real window. Even when you climb into the driver seat, it takes about 25 milliseconds for the engine to actually swap locality over to you. Stepping back out is the same — ~25 ms before the server registers there's no driver anymore.

That window is enough to do useful work. Arma 3 has a command called `moveOut` which kicks remote units out of vehicles, and `moveInDriver` which puts you in. Stitch them together with a couple of `uiSleep` calls and you can briefly grab locality on a vehicle you don't own, run whatever local-only command you want, and step back out before anyone notices.

---

```sqf
_runOnAllVehicles = {
    _vehicle = vehicle player;

    //--- _vehicle is the local vehicle we are in
    _vehicle setVelocity [0, 0, 400];
};

player allowDamage false;
{
    moveOut (driver _x);
    uiSleep 0.025;
    player moveInDriver _x;
    uiSleep 0.025;
    call _runOnAllVehicles;
    moveOut player;
} forEach vehicles;
player allowDamage true;
```

For each vehicle: kick the current driver out, wait 25 ms, jump in, wait another 25 ms for locality to swap, run whatever you want as the now-local owner, then move yourself back out. `allowDamage false` keeps the chair-shuffling from killing you. The body of `_runOnAllVehicles` here just punts every vehicle straight up at 400 m/s, which is funny but obviously not the point — drop any local-only command in there and it'll execute against a vehicle the server otherwise owns.

---

Manipulating vehicle locality this way should never really need to be used in earnest. It's just a neat side-effect I came across while exploring some bugs in the game's networking.

Anyway, that's it.
