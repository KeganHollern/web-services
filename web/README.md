# Web

React + TypeScript SPA built with Vite, styled with Tailwind CSS and shadcn/ui components.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is written to `dist/`, which the Go server serves as a static SPA.

## Pages

| Page | Path | Description |
| --- | --- | --- |
| home | `/` | Homepage |
| blog | `/blog` | Personal blog powered by MDX |
| secret | `/secret` | Zero-trust secret sharing with client-side encryption |
| edit | `/edit` | Collaborative markdown editor |
| swap | `/swap` | Fee-less Uniswap frontend + Aave lending |
| share | `/share` | Zero-trust P2P screen sharing via WebRTC |
| ping | `/ping` | Ping game (WIP) |
